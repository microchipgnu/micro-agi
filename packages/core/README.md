# react-crewai

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.0.14. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.

---

- Use Mintbase Wallet
- Use R3F for agents in games
- Run with Ink on the server side
- WebContainers
